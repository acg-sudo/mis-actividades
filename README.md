# 🎮 Repositorio de Juegos Educativos - Taller de Informática EPA Sedaví

¡Bienvenido/a al espacio central de actividades prácticas e interactivas de nuestro taller de informática! Este repositorio aloja los recursos educativos técnicos desarrollados para complementar las unidades didácticas y que están integrados en nuestro blog cerrado de clase.

Todos los simuladores y juegos aquí listados han sido programados exclusivamente utilizando tecnologías web nativas (**HTML5, CSS3 y JavaScript Vanilla**), garantizando ligereza, rendimiento instantáneo y compatibilidad multiplataforma sin dependencias de terceros.

## 🗂️ Índice de Actividades Disponibles

Aquí puedes acceder a los recursos didácticos individuales y consultar su documentación de desarrollo:

1. 🖱️ **[Desafío: Clasificación de Periféricos](./juego-perifericos/)**
   * *Objetivo:* Repasar y consolidar el concepto de hardware clasificando dispositivos aleatorios en periféricos de entrada, salida o mixtos (E/S).
2. 🔧 **[Simulador: Montaje de PC desde Cero](./simulador-montaje-pc/)**
   * *Objetivo:* Aprender la arquitectura de un ordenador mediante un ensamblaje secuencial guiado y lógico en tres fases (Placa Base, Torre y Periféricos).

## 🔌 Instrucciones de Implementación en Blogger

Para embeber cualquiera de estos simuladores dentro del blog protegido de Blogger una vez activado **GitHub Pages** en este repositorio, edita tu entrada en **Vista HTML** e inserta la etiqueta correspondiente:

```html
<iframe src="https://tu-usuario.github.io/mis-actividades/simulador-montaje-pc/" 
        width="100%" 
        height="620px" 
        style="border:none; border-radius:15px; box-shadow: 0 10px 25px rgba(0,0,0,0.08);">
</iframe>
