# Desafío: Clasificación de Periféricos 🖥️🖱️🔊

¡Bienvenido/a al repositorio del juego interactivo de clasificación de periféricos! Este proyecto ha sido desarrollado como material educativo e interactivo para los alumnos del **Taller de Informática de la EPA Sedaví**.

El objetivo del juego es ayudar a los estudiantes a repasar y consolidar el concepto de periféricos de un ordenador, clasificándolos en tres categorías principales: **Entrada**, **Salida** y **Mixtos (Entrada/Salida)**.

## 🚀 Características del Proyecto

* **Selección aleatoria:** En cada partida se seleccionan 4 periféricos al azar de cada categoría para que cada intento sea un reto diferente.
* **Interactividad "Drag and Drop":** Mecánica intuitiva de arrastrar y soltar las fichas en sus cajas correspondientes.
* **Validación visual:** Al pulsar "Comprobar respuestas", el juego muestra en verde los aciertos y en rojo los errores, permitiendo corregirlos fácilmente.
* **Modo Profesor integrado:** Incluye un acceso oculto mediante una "llave secreta" 🔑 en el pie de página para resolver el tablero automáticamente con fines demostrativos.
* **Lluvia de confeti:** ¡Premio visual integrado en CSS y JS para cuando se completa todo de forma correcta!

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructuración semántica del contenido.
* **CSS3:** Diseño adaptativo (*responsive*), paleta de colores moderna y animaciones nativas para los efectos.
* **JavaScript (Vanilla):** Lógica del juego, manejo de eventos de arrastre (*Drag and Drop*), aleatoriedad y validación.

## 📦 Instalación y Despliegue

Para integrarlo en cualquier plataforma (como Blogger, Moodle, etc.), solo necesitas alojar el archivo `index.html` y la carpeta de imágenes `Perifericos/` con los nombres de archivo correspondientes especificados en el código.

Si usas **GitHub Pages**:
1. Activa GitHub Pages en los ajustes de este repositorio (`Settings` > `Pages`).
2. Selecciona la rama `main` (o `master`) y la carpeta `/root`.
3. ¡Listo! Copia el enlace generado y utilízalo en un `<iframe>` dentro de tu blog de Blogger.
